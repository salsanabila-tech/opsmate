# OpsMate

OpsMate adalah aplikasi mobile untuk membantu bisnis jasa servise mengelola pelanggan, work order, teknisi, progres pekerjaan, dan dokumentasi pekerjaan.

## Teknologi

### Mobile

- React Native
- Expo
- TypeScript

### Backend

- Node.js
- Express
- TypeScript
- PostgreSQL
- Prisma ORM
