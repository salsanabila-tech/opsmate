# OpsMate

## 1. Latar Belakang

OpsMate adalah aplikasi mobile untuk membantu bisnis jasa servis mengatur pelanggan, pekerjaan teknisi, jadwal, status pekerjaan, dan penggunaan spare part.

## 2. Masalah

Saat ini pekerjaan servis sering dicatat melalui WhatsApp atau buku.
Akibatnya:

- Pekerjaan dapat terlupakan.
- Pemilik sulit mengetahui progres teknisi.
- Riwayat servis pelanggan tidak tersimpan dengan baik.
- Penggunaan spare part sulit dilacak.
- Teknisi kesulitan melihat daftar tugas.

## 3. Tujuan

- Menyimpan permintaan servis pelanggan.
- Membagikan pekerjaan kepada teknisi.
- Memantau status pekerjaan.
- Menyimpan riwayat pekerjaan.
- Mencatat spare part yang digunakan.

## 4. Target Pengguna

- Owner atau admin bisnis servis.
- Teknisi.
- Pelanggan.

## 5. Platform

- Aplikasi Mobile React Native.
- Rest API Node.js dan Express.
- Database PostgreSQL
